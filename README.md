## ESP32 Round Clock - HAOS Smart Knob
- ESP32-S3 Round Display (1.28" GC9A01) designed to connect to Home Assistant OS over MQTT and REST API.
- [This project was built around the Elecrow ESP32 Rotary Touch Panel.](https://www.elecrow.com/crowpanel-1-28inch-hmi-esp32-rotary-display-240-240-ips-round-touch-knob-screen.html)
- See DOCUMENTATION.md for full setup instructions and features.
